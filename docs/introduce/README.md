# JaB-UI

JaB-UI 是一个易扩展的 UI 框架，目前提供了一些常用组件（不定期更新），适合 PC 端和移动端使用。

## 现已支持

Button 、Input 、Grid 、Layout 、Toast 、Tabs 、Popover、Collapse
